# 📢 ระบบเรียกชื่อเช็คชื่อด้วยเสียง
**Voice Attendance System — Google Apps Script**

ระบบเรียกชื่อนักเรียนแบบอัตโนมัติ เชื่อมต่อ Google Sheets โดยตรง รองรับการตอบด้วยเสียง (มา / ขาด / ลา) และบันทึกผลแยกรายวิชา

---

## ✨ ฟีเจอร์หลัก

- 📢 **เรียกชื่อทีละคน** — ระบบเรียกชื่อตามลำดับ รอการตอบจากนักเรียน
- 🎤 **รับเสียงตอบ** — นักเรียนพูด "มาครับ / ลาค่ะ / ขาด" ระบบจับและบันทึกอัตโนมัติ
- ✋ **ยืนยันด้วยตนเอง** — กดปุ่ม มา / ขาด / ลา ได้เสมอ
- 📚 **แยกรายวิชา** — อ่านรายชื่อจาก Sheet แต่ละวิชา บันทึกแยก Sheet ต่างหาก
- 📊 **บันทึกลง Google Sheet** — สร้าง Sheet `[บันทึก] วิชา | วันที่` อัตโนมัติทุกวัน
- 📅 **ดูประวัติย้อนหลัง** — เลือกดูสรุปจาก Sheet บันทึกวันที่ต่างๆ
- ⬇️ **Export CSV** — ดาวน์โหลดผลเช็คชื่อได้ทันที
- 📱 **รองรับมือถือ** — UI ปรับตาม screen size

## 📋 โครงสร้าง Google Sheet ที่รองรับ

```
Spreadsheet (เช่น "สมุดเช็คชื่อ 1/69")
│
├── คณิตศาสตร์ ม.3       ← Sheet วิชา
│   ├── A1: เด็กชายกฤษณะ เป้าเงิน
│   ├── A2: เด็กชายกิติภูมิ ถาชินเลิศ
│   └── ...
│
├── คณิตศาสตร์ ม.4       ← Sheet วิชา
├── ภาษาไทย ม.3          ← Sheet วิชา
│
└── [บันทึก] คณิตศาสตร์ ม.3 | 17/06/2568  ← สร้างอัตโนมัติ
    ├── A1: เวลา | B1: ชื่อ-นามสกุล | C1: สถานะ | D1: หมายเหตุ
    ├── A2: 08:30:15 | เด็กชายกฤษณะ | ✓ มา | มาครับ
    └── ...
```

> **ข้อกำหนด:**
> - ชื่อนักเรียนอยู่ **คอลัมน์ A แถว 1** (ไม่มี header)
> - ชื่อ Sheet = ชื่อวิชา (ระบบจะแสดงเป็นปุ่มให้เลือกอัตโนมัติ)
> - Sheet ที่ขึ้นต้นด้วย `[บันทึก]` คือ Sheet ระบบ จะไม่แสดงเป็นวิชา

## 🚀 วิธีติดตั้ง

### ขั้นที่ 1 — เตรียม Google Sheet
1. เปิด Spreadsheet ที่มีรายชื่อนักเรียน
2. ตรวจสอบว่าชื่อนักเรียนอยู่ **คอลัมน์ A แถว 1** ของแต่ละ Sheet วิชา
3. คัดลอก **Spreadsheet ID** จาก URL:
   ```
   https://docs.google.com/spreadsheets/d/[ ID อยู่ตรงนี้ ]/edit
   ```

### ขั้นที่ 2 — สร้าง Apps Script Project
1. เปิด [script.google.com](https://script.google.com) → **"โปรเจกต์ใหม่"**
2. ตั้งชื่อโปรเจกต์ เช่น `ระบบเรียกชื่อเช็คชื่อ`

### ขั้นที่ 3 — วางโค้ด

**Code.gs**
1. ลบโค้ดเดิมออกทั้งหมด
2. วางเนื้อหาจากไฟล์ `Code.gs`
3. แก้ Spreadsheet ID บรรทัดที่ 6:
   ```javascript
   const SHEET_ID = 'วาง_ID_ของคุณ_ที่นี่';
   ```
4. ถ้ารายชื่อ**ไม่มี header** และเริ่มแถว 1 ไม่ต้องแก้อะไร (ค่าเริ่มต้น `DATA_START = 1`)

**Index.html**
1. คลิก **➕ เพิ่มไฟล์** → เลือก **HTML**
2. ตั้งชื่อ **`Index`** (ตัว I พิมพ์ใหญ่ ห้ามมีนามสกุล .html)
3. ลบโค้ดเดิม วางเนื้อหาจากไฟล์ `Index.html`

### ขั้นที่ 4 — Deploy เป็น Web App
1. คลิก **Deploy** → **"New deployment"**
2. คลิกไอคอน ⚙️ → เลือก **"Web app"**
3. ตั้งค่า:
   - **Execute as**: Me (บัญชีของฉัน)
   - **Who has access**: Only myself (ใช้คนเดียว) หรือ Anyone
4. คลิก **Deploy** → อนุมัติสิทธิ์ → คัดลอก **Web App URL**

> ⚠️ **ทุกครั้งที่แก้ไขโค้ด** ต้อง Deploy ใหม่:
> Deploy → Manage deployments → ✏️ แก้ไข → Version: **New version** → Deploy

### ขั้นที่ 5 — ตั้งค่าไมโครโฟนใน Chrome
เนื่องจากโดเมน `script.google.com` ต้องได้รับอนุญาตไมโครโฟนก่อน:

1. เปิด `chrome://settings/content/microphone`
2. ส่วน **"อนุญาต"** → กด **"เพิ่ม"**
3. วาง URL ของ Web App (เช่น `https://script.google.com`) → เพิ่ม
4. รีโหลดเว็บ → กด ▶ → Chrome จะถามขอ mic → กด **"อนุญาต"**

## 🎯 วิธีใช้งาน

1. **เลือกวิชา** จากแถบด้านบน — รายชื่อโหลดจาก Sheet อัตโนมัติ
2. **กดปุ่ม ▶** — ระบบเริ่มเรียกชื่อทีละคน
3. **นักเรียนตอบ:**
   | พูดว่า | ระบบบันทึก |
   |--------|-----------|
   | "มาครับ" / "มาค่ะ" / "ครับ" / "ค่ะ" | ✓ มา |
   | "ลาครับ" / "ลาค่ะ" / "ป่วย" | 🏥 ลา |
   | "ขาด" / "ไม่มา" | ✗ ขาด |
4. **ถ้าไม่มีการตอบ (8 วินาที):**
   - กด **"ข้าม"** → นับว่าขาด และเรียกคนถัดไป
   - หรือกดปุ่ม **มา / ขาด / ลา** ด้วยตนเอง
5. **ดูผล** ได้จากแท็บ "บันทึกวันนี้" และ Google Sheet โดยตรง

## ⚙️ ค่าที่ปรับได้ใน Code.gs

| ค่า | ความหมาย | ค่าเริ่มต้น |
|-----|---------|-----------|
| `SHEET_ID` | Spreadsheet ID | `''` (ใช้ Sheet ที่ผูก script) |
| `NAME_COL` | คอลัมน์ที่เก็บชื่อ (A=1, B=2) | `1` |
| `DATA_START` | แถวแรกที่มีข้อมูล | `1` |
| `LOG_MARK` | prefix ของ Sheet บันทึก | `'[บันทึก]'` |

## ❓ แก้ปัญหาที่พบบ่อย

**เว็บไม่โหลด / ค้างหน้าขาว**
→ Deploy ใหม่ด้วย New version ทุกครั้งที่แก้โค้ด

**ไมโครโฟนไม่ทำงาน / ไม่มี popup ขึ้น**
→ เพิ่ม URL ใน `chrome://settings/content/microphone` ตามขั้นที่ 5

**ชื่อวิชาในแถบด้านบนกดไม่ได้**
→ ชื่อ Sheet มีอักขระพิเศษ — ระบบใช้ event listener แล้ว ควรทำงานได้กับทุกชื่อ

**กดปุ่มแล้วไม่มีอะไรเกิดขึ้น**
→ ตรวจสอบว่าได้ Deploy ไฟล์ `Index.html` ล่าสุดแล้ว (ชื่อไฟล์ต้องเป็น `Index`)

**Speech API ไม่ทำงาน**
→ ต้องใช้ **Chrome หรือ Edge** เท่านั้น (Safari และ Firefox ไม่รองรับ)
